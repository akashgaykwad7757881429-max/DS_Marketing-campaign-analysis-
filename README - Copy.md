**# Marketing Campaign Analysis using Power BI**



\## Project Overview



This project analyzes customer income based on education level using Power BI visualization.



The project helps to:



1\. Analyze customer income distribution

2\. Compare income based on education level

3\. Visualize marketing campaign data using Power BI



\## Technologies Used



• Power BI  

• CSV Dataset  

• Data Visualization  



\## Files



• marketing\_campaign\_data.csv → Marketing campaign dataset  

• marketing\_data\_dictionary.csv → Dataset column details  

• Marketing\_Campaign\_Analysis.pbix → Power BI project file  

• Screenshot.png → Output chart screenshot  



\## Visualization Used



• Bar Chart → Sum of Income by Education



\## Insights



• Customers with \*\*Graduation\*\* education level have the highest total income.  

• Customers with \*\*Basic\*\* education level have the lowest total income.  

• Education level impacts income distribution significantly.



\## Run Project



1\. Open Power BI Desktop  

2\. Load `marketing\_campaign\_data.csv` dataset  

3\. Select \*\*Bar Chart visualization\*\*  

4\. Add \*\*Education\*\* to Y-axis  

5\. Add \*\*Income\*\* to X-axis  

6\. Generate the visualization



\## Output



The final output is a \*\*Bar Chart showing Sum of Income by Education\*\*.
